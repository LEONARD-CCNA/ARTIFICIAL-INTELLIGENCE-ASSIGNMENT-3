2 · The method

Three models, written from first principles in Lisp

Each model consumes the same 19 features per month: current fuel prices (diesel, petrol, kerosene, composite index, petrol/diesel ratio), diesel lags 1–3, momentum terms, CPI, year-on-year inflation, USD/KES, the commodity's own lags 1–3, and seasonal sin/cos encodings. Scalers are fitted on training months only — no leakage. Evaluation uses a strict temporal split (60 train / 15 test) plus walk-forward validation, where every model is retrained each test month.

Linear Regression (Ridge)

Ordinary least squares with an L2 penalty (λ = 1.0), solved via the normal equations and Gaussian elimination with partial pivoting. The interpretability baseline.

Random Forest

200 CART regression trees grown on bootstrap samples with feature subsampling (mtry = √p), max depth 10, min leaf 2. Provides impurity-based feature importance.

LSTM Network

Input/forget/output gates and memory cell with a 3-month lookback, trained by hand-derived backpropagation-through-time with the Adam optimizer. 8 hidden units — sized to the data.

6 · Findings

What the real data taught us

  • Models beat the naive baseline, modestly. Walk-forward RMSE: Random Forest 3.91 vs naive 4.22 on maize; Linear Regression 11.03 vs 12.75 on vegetable oil; LSTM 0.80 vs 0.89 on bread. Monthly prices are strongly autoregressive, so persistence is a demanding yardstick.
  • Negative R² on maize/bread is a property of the test window, not a broken model. Bread barely moved during testing (tiny variance denominator: 1.5% MAPE still scores R² < 0), and the window contains the COVID-19 regime shift absent from training. MAPE and RMSE are the informative metrics here.
  • FX and CPI, not raw diesel, carry the most signal — diesel matters through lags (lag-3 for oil) and jointly with the macro variables, exactly the conditional transmission the literature describes.
  • Small data favors simple models. With 60 training months, ridge regression is the most consistent; the first LSTM (2×64/32 units as designed in Assignment 2) memorized the training set and had to be reduced to 8 units — a live demonstration of the bias–variance trade-off.
  • Walk-forward > fixed split for a deployed system. Retraining monthly as EPRA/WFP publish new prices improved every model — this is how the tool would actually operate.

Honest deviations from the Assignment 2 write-up

  • Bread replaces sugar — sugar has no sustained Nairobi series in the WFP data; bread is the third fuel-sensitive staple with 76/78 months of coverage.
  • Window 2014–2020 instead of 2019–2023 — Nairobi fuel monitoring in the WFP dataset ends Dec 2020; 78 observations beat 60.
  • LSTM reduced to one layer of 8 units — the write-up's stacked 64/32 network overfit ~60 samples.
